# OrgPulse Sample Screenshots

## scale-center-sample.png

A realistic Scale Center Org Performance screenshot used for the onboarding flow.

**Source:** Development org owned by the OrgPulse maintainer. Contains no customer data, no identifying org metadata, and no sensitive information — only aggregated performance metrics visible on any Scale Center dashboard.

**Characteristics:**
- 23 successful logins, 0 failed logins
- 0 concurrent Apex errors, 0 concurrent UI errors
- 2 row lock errors
- 0 callout errors (callout charts empty)
- Visible CPU spikes at ~02:00, ~07:00, and ~18:00 UTC
- Request time spikes correlating with CPU activity
- Periodic request volume patterns suggesting batch activity

**Why this sample was chosen:**
The screenshot contains enough signal variety to demonstrate multiple OrgPulse features (counter extraction, chart pattern detection, signal-to-matrix mapping) while showing realistic production-like values rather than contrived extremes.
